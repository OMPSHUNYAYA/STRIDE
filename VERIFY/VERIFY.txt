1. VERIFY THE FROZEN HTML

Expected file:

demo/STRIDE_v1_1_3.html

Windows:

certutil -hashfile demo\STRIDE_v1_1_3.html SHA256

macOS / Linux:

sha256sum demo/STRIDE_v1_1_3.html

Compare the calculated SHA-256 value with:

VERIFY/FREEZE_DEMO_SHA256.txt

The values must match exactly.

If the values do not match, stop verification.


2. START A LOCAL SERVER

Run from the repository root.

Windows:

19. VERIFY PAUSE BEHAVIOR

Reset the observatory.

Select:

RESOLVED

Press:

EVALUATE

Press:

EXECUTE

Press:

PAUSE

Confirm:

the phase displays MOTION PAUSED

the control changes from PAUSE to RESUME

motion animation stops

phase progression stops

the interface remains responsive

reset remains available

the carried-object state remains coherent

Run:

STRIDE.runtimeStatus()

Expected:

paused = true

animation_active = false

phase_timer_active = false


20. VERIFY RESUME REVALIDATION

While paused, change a mandatory condition to an invalid value.

Example:

set HUMAN ZONE to Occupied

Press:

RESUME

Confirm:

motion does not resume

the runtime remains paused

Restore the mandatory condition.

Example:

set HUMAN ZONE to Clear

Press:

RESUME

Confirm:

STRIDE automatically revalidates the current structure

motion resumes only when the structure is again RESOLVED

Expected invariants:

resume_request + invalid_structure -> remain_paused

PAUSED + renewed_validity -> RESUME

A separate manual EVALUATE step is not required before pressing RESUME.

24. FAILURE CONDITIONS

Verification fails if any of the following occur:

the HTML checksum does not match

the reported version is not 1.1.3

a declared audit returns allPass = false

a non-resolved scenario creates a Motion Contract

an unsafe scenario becomes admitted

pause does not stop execution

resume occurs without renewed structural validity

completion leaves an active scheduler

the qualification does not complete 346 cases

unsafe_admissions is greater than 0

equivalent structure produces inconsistent admission


25. VERIFICATION RESULT

A successful result establishes reproduction of the bounded STRIDE v1.1.3 reference behavior.

It does not establish:

certified functional safety

physical robot validation

actuator-level enforcement

measured stopping performance

secure production cryptography

regulatory approval

production deployment readiness
